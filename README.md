## 安装

这个项目使用 [go](https://golang.org/)请确保你本地安装了它们。

```sh
$ tar -C /usr/local -xzf go1.4.linux-amd64.tar.gz
$ export PATH=$PATH:/usr/local/go/bin
```

## 使用说明

```sh
$ go build -o monkey main.go
$ ./monkey
```
